## BeanYard :coffee:
#### What is BeanYard?
<p>나머지 98.9%는 사용되지 못한채 버려지곤 합니다. 아메리카노 한 잔을 만들기 위해 약 15g의 커피원두가 사용되는데, 이 중 14.97g 가 버려지는 셈입니다.</p>
<p>커피박은 현재 폐기물 쓰레기로 배출되고 있지만, 재활용 가치가 높은 유기성 자원입니다.</p>
<p>그래서 우리는 커피박을 재활용 할 수 있는 방안에 대해 생각해보았습니다.</p>
